
## 5. 使用方法

### 5.1 初始化

1. 确保串口硬件已正确初始化
2. 在主循环中调用 `vofa_draw_graphical` 函数发送数据
3. 在串口接收中断中调用 `vofa_set_data` 函数处理接收的数据

### 5.2 发送数据

```c
// 在主循环中调用
void main(void) {
    while(1) {
        // 发送左电机PID数据
        vofa_draw_graphical(PID_LEFT_MOTOR);
        
        // 发送角度PID数据
        vofa_draw_graphical(PID_ANGLE);
        
        // 延时
        delay_ms(10);
    }
}
```

### 5.3 接收数据

```c
// 在串口接收中断中调用
void uart_rx_interrupt_handler(void) {
    uint8_t data;
    if(uart_query_byte(UART_INDEX, &data)) {
        vofa_set_data(data);
    }
}
```

## 6. 数据类型标志

| 标志 | 含义 | 数据通道 |
|------|------|----------|
| `PID_LEFT_MOTOR` | 左电机速度环PID | current, target, kp, ki |
| `PID_ANGLE` | 角度环PID | current, target, kp, kd |
| `PID_RIGHT_MOTOR` | 右电机速度环PID | 预留 |
| `PID_TURN` | 转向环PID | 预留 |

## 7. 参数调整

通过VOFA+上位机发送以下格式的指令来调整参数：

| 指令格式 | 功能 | 对应变量 |
|----------|------|----------|
| `P1=value` | 设置左电机速度环P参数 | `speed_left.Kp` |
| `I1=value` | 设置左电机速度环I参数 | `speed_left.Ki` |
| `T1=value` | 设置左电机目标速度 | `speed_left.Target` |
| `P2=value` | 设置右电机速度环P参数 | `speed_right.Kp` |
| `I2=value` | 设置右电机速度环I参数 | `speed_right.Ki` |
| `T2=value` | 设置右电机目标速度 | `speed_right.Target` |

## 8. VOFA+ 上位机设置

1. 打开VOFA+软件
2. 选择 "端口" 标签页，设置正确的串口参数（波特率等）
3. 选择 "协议" 标签页，选择 "FireWater" 协议
4. 选择 "波形" 标签页，添加需要显示的通道
5. 点击 "连接" 按钮开始通信

## 10. 常见问题与解决方案

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 无数据显示 | 串口未连接 | 检查串口连接和波特率设置 |
| 数据显示乱码 | 协议选择错误 | 确保VOFA+选择了FireWater协议 |
| 参数调整无效 | 指令格式错误 | 确保指令格式为 "标识符=数值\n" |
| 数据更新缓慢 | 发送频率过低 | 增加发送频率，减少延时 |
| 缓冲区溢出 | 接收数据过快 | 确保及时处理接收到的数据 |
